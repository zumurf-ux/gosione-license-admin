# 공용 라이선스 관리 페이지

## 변경 내용

- 라이선스별 `확인` 버튼을 제거했습니다.
- 검색/필터 영역의 `새로고침` 오른쪽에 `업데이트 팝업` 버튼을 추가했습니다.
- `업데이트 팝업` 버튼을 누르면 등록 기기가 있는 모든 라이선스에 확인 요청을 보냅니다.
- 각 기기 앱이 다음 실행 또는 서버 확인 시 요청을 받아 업데이트 확인을 실행합니다.

## 업로드 방법

GitHub Pages 저장소에는 이 폴더의 `index.html`을 올리면 됩니다.

## 필요 조건

Supabase Edge Function `license-admin`에 `request_device_check` 액션이 적용되어 있어야 합니다.
기존에 개별 확인 버튼이 동작하던 상태라면 추가 서버 수정 없이 사용할 수 있습니다.
